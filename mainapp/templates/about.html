About Netflix AI

About This Recommendation System

Netflix AI is a movie recommendation system built using Django, Python, Machine Learning, and Netflix dataset analysis.

Machine Learning

Uses similarity matrix logic to recommend movies related to the user’s searched movie.

Netflix Dataset

Works with cleaned Netflix movies and TV shows data for better recommendation results.

Django Project

Built with Django templates, static files, Bootstrap, CSS and backend Python logic.

Project Overview

How This System Works

The user enters a movie name. The system finds that movie in the dataset, checks its similarity score with other movies, and returns the most similar movie recommendations.

This project combines data cleaning, feature selection, similarity matrix generation, Django views, templates, and a modern Netflix style frontend.
